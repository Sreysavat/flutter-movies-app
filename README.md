# 🎬 Flutter Movies App

A mobile application built with Flutter that displays movie/news content using API integration.

## 🚀 Features
- Fetch data from REST API
- Display movie/news list
- View details screen
- Clean UI design
- State management (GetX)

## 🛠️ Technologies Used
- Flutter
- Dart
- REST API
- GetX

## 📱 Screens
- Login
- Register
- Home Screen (list of news/movies)
- Details Screen

## ⚙️ Getting Started

### Prerequisites
- Flutter installed
- Android Studio / VS Code

### Run the project
```bash
flutter pub get
flutter run
